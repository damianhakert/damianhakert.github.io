Open Source Git Management Software.

Gitlab is an open source git repository management application. It allows developers to collaborate effectively and securely.
More than 25,000 organizations are using GitLab on-premises. The community site for GitLab and the source code can be found on GitLab.org.

GitLab.com is co-founded by the lead author of GitLab to offer commercial services for GitLab. We offer a subscription, consulting and a free SaaS service called GitLab Cloud.
