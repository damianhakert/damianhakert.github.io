Open Source Git Management Software

Gitlab is open source software to collaborate on code. Manage git repositories with fine grained access controls that keep your code secure. Perform code reviews and enhance collaboration with merge requests. Each project can also have an issue tracker and a wiki. Used by more than 50,000 organizations, GitLab is the most popular solution to manage git repositories on-premises.

GitLab Features

Below are some highlights, a complete list is available on the feature comparison page.

Repository Access

Extended permission system with 5 access levels and branch protection.

Administration

Simplify management by creating groups of projects and teams of users.

Issues

Use the ticketing system included in GitLab or integrate your existing system.

Code Review

Learn from eachother by leaving line comments in the diff.

Wiki

Each project has a wiki backed up by a separate git repository.

Merge Requests

Work in git branches for a more efficient workflow.
