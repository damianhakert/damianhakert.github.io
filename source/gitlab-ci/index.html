GitLab Continuous Integration Server

GitLab CI integrates with your GitLab installation and run tests for your projects..
Login with GitLab account, add project with one click and enjoy running tests.

About

Requirements

GitLab CI is designed for the Linux operating system.

GitLab CI officially supports (recent versions of) these Linux distributions:

  • Ubuntu Linux
  • Debian/GNU Linux

Additionally GitLab CI requires:

  • ruby 1.9.3+
  • MySQL or PostgreSQL
