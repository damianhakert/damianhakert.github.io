GitLab Continuous Integration

Let GitLab test, build, deploy your code today!

Download Documentation

What are the advantages?

  • Easy to set up with included Omnibus packages of GitLab CE / GitLab EE.
  • Beautiful interface with a clear menu structure.
  • Performant and stable, as tests run distributed on separate machines.
  • Receive test results faster with each commit running in parallel on multiple jobs.
  • Free to use and completely open source

What's up? Now you know.

  • Adding a project and setting up the hooks is accomplished in one click.
  • One click to see the real-time build log output.
  • You can see the test results inside the merge request of GitLab.

Fully integrated with GitLab

  • GitLab CI is fully integrated with GitLab.
  • No need for separate authentication and all your projects are immediately available.

Requirements

GitLab CI is officially supported on these Linux distributions:

  • Ubuntu
  • Debian/GNU
  • CentOS
  • Red Hat Enterprise Linux
  • Scientific Linux
  • Oracle Linux
