Contributing to GitLab

Contributing Guidelines

If you want to contribute to GitLab, make sure to read and follow the contributing guidelines.

Contributing Process

Contributors and maintainers should know what to expect form each other in the contribution process. This is described here.

Feedback and Suggestions Forum

Do you have any feedback, a suggestion or want to hear what others have to say? Find it in the feedback forums.

Sponsor the development of new features

If you want us to implement a certain feature, you can sponsor us to do so.
