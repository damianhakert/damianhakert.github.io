GitLab.com is Gitlab as a Service

GitLab.com can host your software projects for free. It offers (private) repositories, issue tracking, wiki's and continuous integration. It runs GitLab Enterprise Edition and GitLab CI. You don't have to install anything, just sign up below for a free account.

Sign Up

Advantages over GitHub

  • Unlimited repositories
  • Unlimited private collaborators
  • Unlimited disk space *
  • Completely free, no credit card required
  • Free Continuous Integration (CI) on ci.gitlab.com

* For performance reasons there is a soft limit of 1GB per repository

Support package

Free subscribers can use the GitLab.com Support Forum if they have questions. If you purchase GitLab.com Bronze Support you can email support directly for timely, personal and private answers. This costs $9.99 per user per year for next-business-day response time and is available in packs of 20 users.
