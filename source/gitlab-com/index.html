GitLab.com is Gitlab as a Service

GitLab.com can host your software projects for free. It offers (private) repositories, issue tracking, wiki's and continuous integration. It runs GitLab Enterprise Edition and GitLab CI. You don't have to install anything, just sign up below for a free account.

Sign Up

Advantages over GitHub

  • Unlimited repositories
  • Unlimited private collaborators
  • Unlimited disk space *
  • Completely free, no credit card required
  • Free Continuous Integration (CI) on ci.gitlab.com

* For performance reasons there is a soft limit of 1GB per repository

Support package

Free subscribers can use the GitLab.com Support Forum if they have questions. If you purchase GitLab.com Bronze Support you can email support directly for timely, personal and private answers. This costs $99 per year for one named contact and a next-business-day response time.

Availability

GitLab.com is monitored 24/7. For updates follow @gitlabstatus or see our status page.

Exporting

GitLab.com does not currently allow you to easily export all your data to your own GitLab instance. Of course you can git clone the repositories and import them with a rake task. And since the wiki is backed by a git repo you can clone that as well. But these are manual actions and for the issues you'll have to use the issue API. So if you plan on eventually using GitLab on your own server please consider doing so from the start. Of course we are accepting merge requests to improve this situation.
