Exporting

GitLab.com does currently not allow you to easily export all your date to your own GitLab instance. Of couse you can git clone the repositories and import them with a rake task. And since the wiki is backed by a git repo you can clone that as well. But these are manual actions and for the issues you'll have to use the issue API. So if you plan on eventually using GitLab on your own server please consider doing so from the start.
