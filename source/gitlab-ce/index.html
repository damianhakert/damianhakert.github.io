Community Edition

GitLab is open source software to collaborate on code.
Create projects and repositories, manage access and do code reviews.

Downloads Source code

GitLab Features

Repository Access

Extended permission system with 5 access levels and branch protection.

Administration

Simplify management by creating groups of projects and teams of users.

Issues

Use the ticketing system included in GitLab or integrate your existing system.

Code Review

Learn from eachother by leaving line comments in the diff.

Wiki

Each project has a wiki backed up by a separate git repository.

Merge Requests

Work in git branches for a more efficient workflow.

GitLab allows you to

  • keep your code secure on your own server
  • manage repositories, users and access permissions
  • communicate through issues, line-comments and wiki pages
  • perform code review with merge requests

GitLab is

  • powered by Ruby on Rails
  • completely free and open source (MIT license)
  • used by more than 50.000 organizations to keep their code secure

Requirements

  • Ubuntu/Debian
  • ruby 1.9.3+
  • git 1.7.10+
  • redis 2.0+
  • MySQL or PostgreSQL

** More details are in the requirements doc
