I want to install GitLab on
  • Ubuntu 12.04
  • CentOS 6.4
  • Debian 7

Download and unpack the package and dependencies to your server

wget https://downloads-packages.s3.amazonaws.com/ubuntu-12.04/gitlab_6.9.2-omnibus.1-1_amd64.deb
sudo apt-get install openssh-server
sudo apt-get install postfix # sendmail or exim is also OK
sudo dpkg -i gitlab_6.9.2-omnibus.1-1_amd64.deb

Install and start GitLab

sudo gitlab-ctl reconfigure
