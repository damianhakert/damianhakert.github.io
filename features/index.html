Community Edition and Enterprise Edition
GitLab comes in two editions: the Community Edition and the Enterprise Edition. The Community Edition can be obtained via GitLab.org. The Enterprise Edition is only available to subscribers of GitLab.com. The Enterprise Edition includes all the features of the Community Edition and two additional features. The first feature is synchronizing LDAP groups to GitLab. This is in addition to the LDAP user synchronizing that is included in both editions. The second feature is giving the members of another group access to a project. This is in addition to the normal access for project members and for members of the group the project is in. Below is a short list of some of the features in each edition. For a visual overview of GitLab please see the screenshots on GitLab.org. To obtain GitLab Enterprise Edition please sign up for a subscription.

Git
