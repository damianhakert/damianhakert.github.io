Features
Gitlab.org is quickly becoming popular because it is easier to install and has all the important features such as merge requests. We think that the workflow enabled by merge requests is making software development a lot more fun.

Community
Gitlab.org is open source software and can be used and modified free of charge. It is actively maintained, has an active community and is in active use within many companies such as Thomson Reuters. Gitlab is in the top 50 most popular open source projects.

Git
Gitlab is based on the Git version control system. A good reference is the free Pro Git ebook and Clay Shirky gave a good presentation about the potential of decentralized version control to transform society.
