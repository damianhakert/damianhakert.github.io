Packages (beta, do not use in production)
These packages contain GitLab and all its depencies (PostgreSQL, Redis, Nginx, Unicorn, etc.). They are made with with omnibus-gitlab that you can use to create your own package. These packages currently support a reduced selection of GitLab's normal features. It is not yet possible to create/restore application backups or to use HTTPS, for instance.

Platform Download
Ubuntu 12.04 LTS 64bit gitlab_6.6.0.pre1.omnibus.ubuntu.12.04_amd64.deb
CentOS 6.5 64bit
